# Star Wars Who's Who

The objective of the exercise is to improve the Star Wars "Who's Who" application 

Thanks to this super usefull application you can easily browse through all the characters data available in the famous [StarWars API](https://swapi.dev/)

This API contains all the characters, vehicles, systems and movie from the StarWars universe. 

Thanks to this application you can easily search through the list of characters, filter them by gender and system of origin and even add a profile picture

# Launching the App

Prequisite : you need to have Docker installed version >= 28.4

```
docker compose run --service-ports app
```

Once Docker has started 

```
node@24f4d4cbe6de:~/app$ yarn install
node@24f4d4cbe6de:~/app$ yarn dev
```

You should have the app running on `https://localhost:3000`
